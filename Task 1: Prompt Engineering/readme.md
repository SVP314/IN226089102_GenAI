# Mini Prompt Engine (LangChain)

## Overview

A modular prompt system built using LangChain to replace hardcoded prompts with reusable templates.

## Features

* PromptTemplate for dynamic prompts
* Multi-input handling (topic, audience, tone)
* Prompt variations (teaching, interview, storytelling)
* ChatPromptTemplate with role-based prompts
* Input validation layer
* Reusable template design

## Tech Stack

Python, LangChain, Google Colab

## Pipeline

User Input → Validation → Prompt Template → Output

## Run

Install dependencies and run the notebook:

```bash
pip install langchain
```

## Output Example

```text
Explain Neural Networks for beginner in a fun storytelling style
```
